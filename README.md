# Multiplication-Table-Generator
Multiplication Table Generator
Description

This is a simple Python menu-driven program that generates the multiplication table of any number till the range entered by the user.

The program uses:

Functions
Loops
User Input
Conditional Statements
Features
Generate multiplication table for any number
User chooses the multiplication range
Menu-driven interface
Exit option included
Concepts Used
def functions
while loop
for loop
if-elif-else
range()
Formatted strings (f-strings)
How It Works
Program displays a menu.
User selects:
1 → Generate Table
2 → Exit
User enters:
Number for table
Range till multiplication
Program prints the multiplication table.
